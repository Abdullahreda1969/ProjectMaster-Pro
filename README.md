# ProjectMaster-Pro 🚀

نظام إدارة مشاريع متكامل مبني على Node.js و React و MongoDB Atlas.

## ✨ المميزات

- 📊 لوحة تحكم مع إحصائيات حية
- 📁 إدارة المشاريع (إضافة، تعديل، حذف، بحث)
- ✅ إدارة المهام مع تصفية متقدمة
- 👥 إدارة فرق العمل والأعضاء
- 💰 نظام فواتير متكامل

## 🛠️ التقنيات المستخدمة

- **Backend**: Node.js + Express + MongoDB Atlas
- **Frontend**: React + Tailwind CSS
- **Database**: MongoDB Atlas
- **Authentication**: JWT (قريباً)
